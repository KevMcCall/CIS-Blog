Comparison Summary

For performance Acer Nitro 5 has a better performing processor. Microsoft Surface Pro 6 has Windows 10 Pro, which is safer and provides more security than Windows 10 Home that is in the Nitro 5. For displaying graphics the Nitro 5 has a lot faster graphics card and has a bigger screen. The Surface Pro 6 is cheaper and also has a SSD, while the Nitro 5 has a HDD storage. The Acer Nitro 5 is a little bulkier than the Surface Pro 6, but has a lot more processing power and comes equipped with Nitrosense technology which allows the user to control the speed of the fans for whenever the workload is a bit heavy. The Surface Pro 6 can be used as a tablet, which takes advantage of its touch screen interface, or as a notebook using the keyboard attachment, which doubles as a cover for the display, or in studio mode, which enables the use of the Surface Pen and Dial..


All in all, both are great laptops. However I would suggest that if you're looking for a powerful laptop which can handle more tasks as well as gaming, I would go with the Nitro 5. If you're needing something that's a bit more lightweight, portable, using it for basic needs, or graphic design, than I would recommend the Surface Pro 6.

Computer Shoes

Imagine that you're living in the year 2090. Technology has advanced to the point that any inanimate object can function as a computer. Now... Imagine that technology, in your shoes.

Tech Features:

  • Input: The shoes feature a small LCD display which can be used to change the color of the shoes, loosen or tighten the shoes to your preference, display exercise stats, as well as many other settings. It also features a Micro USB port to charge them.

  • Memory: The shoe can record for 30 days worth of data. Every new month is a fresh start. Throughout the month the runner can access all the stored data in a Monthly Statistics tab.

  • Output: Data can be synced to your profile via an app or using the LCD display. They can also play music through small integrated speakers, which are built into the shoes.

  • Control Unit and Arithmetic Logic Unit (ALU): Both are located in the shoes left sole. The control unit tells the ALU what operation to perform based on the data the wearer enters and the ALU then stores the result in an output register.
